# PyroTC enclosure STLs

3D-printed housing for the Waveshare ESP32-S3-Touch-LCD-1.28 pyrometer.

| File | Part |
|------|------|
| `tiretester.stl` | Main body — head + cylindrical handle |
| `cover.stl` | Faceplate over the display/sensor bay |
| `cap.stl` | Screw-on 18650 battery cap with contact spring |

Designed in Fusion 360. Fits MAX6675 breakout, rocker switch, and 18650 in the grip.
